Getting Started

This project is an Angular-based web application that uses PrimeNG for UI components. It is designed to manage various aspects of Correspondence System.

Make sure you have Angular CLI installed before proceeding. If Angular CLI is not installed, you can set it up using the following command:

npm install -g angular/cli

Once you have cloned the repository and set up the required dependencies, the application should run at http://localhost:4200/. You can then start developing your application.

Setting Up the Development Environment
  1. Ensure that you have Node.js (v18 or higher) and npm installed on your machine. You can download them from Node.js official site.
  2. Clone the project repository to your local machine using the command: 
    git clone https://github.com/InesE21/correspondenceSystem-Front.git
  3. Navigate to the project folder: 
    cd correspondence-system
  4. Install all required dependencies by running: 
    npm install
Running the Application
  1. Start the development server with the command: 
    ng serve
  2. Open a browser and navigate to http://localhost:4200/.
  3. The application will automatically reload when you make changes to the source files.
